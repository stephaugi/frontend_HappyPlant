# HappyPlant
A plant care mobile app developed for both IOS and Android. HappyPlant is focused on letting the user keep track of their plants watering schedule.

## Features
* Add a plant and its details to a list of plants.
* Edit and delete plants
* View all plants and see an icon that indicates when it should be watered.
* Log the soil's moisture level in a daily tracker.
* Receive an alert when the plant is ready to water.
* Mark when a plant has been watered.
* Receive recommended watering times based on previous waterings or soil moisture level.
* View previous waterings and soil moisture logs on a calendar view.

## Usage
To run this app:
1. Install all dependencies.

```npm install```

2. Run the app using the following command in the project.
```npx expo start```

3. This will run the app on a simulator. If you'd like to run the app on a mobile device, you can install the Expo Go app on your mobile app store and scan the generated QR code to use the app.
